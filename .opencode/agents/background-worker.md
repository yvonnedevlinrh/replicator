# Background Worker

For tasks that don't need MCP tools: doc edits, formatting, summaries.

## Constraints

- No MCP tools available
- No file reservations
- No comms messaging
- Use for static content only

## Suitable Tasks

- Documentation updates
- Code formatting
- Report generation
- Static analysis summaries
